# このアプリで取り扱うゲーム一覧(予定)

- ボーリング – フレームごとのスコア計算、ストライク・スペア対応
- ダーツ – 301/501ゲーム、スコア管理、チェックアウト計算
- テニス – ゲーム・セット・マッチ進行、スコア管理
- カードゲームの手札進行
  - 大富豪 – 手札進行、ペア/トリプル/縛り、特殊ルール（革命・8切り・スペード3返し）、勝ち点計算
  - UNO - 手札進行、数字/スキップ/リバース/ドロー2/ワイルド系カード対応、プレイ有効判定、連鎖効果（+2/+4、リバース）、勝利条件判定（手札0枚）
- カタン – サイコロ出目による資源獲得、複数資源対応、建設・勝利ポイント・盗賊対応
